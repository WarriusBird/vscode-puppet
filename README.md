# Puppet Extension for Visual Studio Code

The Puppet Extension for Visual Studio Code offers rich language support for Puppet DSL and snippets when using [Visual Studio Code](http://code.visualstudio.com).

## Features

### Syntax Keywords
- Puppet DSL Syntax

## Snippets
- augeas
- case
- cron
- define
- exec
- file
- group
- if
- elsif
- else
- node
- package
- service
- user
- yumrepo
- zfs
- zpool
